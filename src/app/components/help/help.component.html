Preguntas Frecuentes

Para buscar un servicio en ProService, simplemente utiliza la barra de búsqueda en la página principal e ingresa el tipo de servicio que necesitas. Se mostrarán una lista de profesionales disponibles.

Una vez que encuentres al profesional adecuado, puedes contactarlo directamente a través de nuestra plataforma. En la página del perfil del profesional, encontrarás la opción para comunicarte con él.

Puedes cancelar un servicio programado contactando al profesional directamente o a través de tu perfil en ProService. Es importante comunicarte con anticipación para evitar inconvenientes.

Si no estás satisfecho con el servicio, puedes comunicarte con nosotros a través de nuestro soporte para resolver cualquier problema o dejar una valoración honesta en la página del profesional.

El pago por los servicios contratados se realiza directamente al profesional. ProService no interviene en el proceso de pago; es responsabilidad del usuario y el profesional acordar los términos de pago.

Si el profesional no llega a la cita acordada, te recomendamos contactar con él directamente para obtener una explicación o reprogramar la cita. Si el problema persiste, puedes comunicarte con nosotros para asistencia adicional.

Sí, puedes cancelar una solicitud de servicio después de confirmada. Te recomendamos hacerlo con la mayor antelación posible para evitar inconvenientes tanto para ti como para el profesional.

El método de pago se acuerda directamente entre tú y el profesional. Puedes acordar el pago en efectivo, por transferencia bancaria u otros medios de pago aceptados por el profesional.

Puedes revisar las valoraciones y comentarios de otros usuarios en el perfil del profesional para evaluar su confiabilidad. Además, nuestra plataforma verifica la información de los prestadores para ofrecerte una experiencia segura.

Si surge un problema después de completar el servicio, contáctanos para brindarte asistencia y orientación. También puedes dejar comentarios detallados en el perfil del profesional para informar sobre tu experiencia.
